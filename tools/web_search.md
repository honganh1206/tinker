Search the web for up-to-date information using Brave Search.

WHEN TO USE THIS TOOL:
- When you need current or real-time information beyond your knowledge cutoff
- When the user asks about recent events, news, or current data
- When you need to look up documentation, APIs, or library versions
- When you need to verify facts or find authoritative sources
- When the user explicitly asks to search the web

WHEN NOT TO USE THIS TOOL:
- When you can confidently answer from your existing knowledge
- When the question is about the local codebase (use grep_search or finder instead)
- When the user asks about files or directories (use read_file or list_files instead)

SEARCH TIPS:
- Use specific, targeted search queries for best results
- Include key terms and context in your query (e.g., "Go golang http middleware timeout" rather than just "timeout")
- If the first search doesn't return useful results, try rephrasing the query
- Combine multiple searches for complex topics that span different areas
